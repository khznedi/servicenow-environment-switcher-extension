# ServiceNow Environment Switcher Chrome Extension

A lightweight Chrome extension designed to help ServiceNow developers, testers, and administrators quickly switch between environments and investigate records faster.

## Features

- Auto-detect ServiceNow instance environment
- Visual environment indicator (color frame)
- Quick navigation between DEV / UAT / PROD
- Sys_id detection from current page
- Right-click context troubleshooting dashboard
- User impersonation helper (planned)

## Why I built this

Working across multiple ServiceNow environments daily creates friction during troubleshooting and navigation. This extension reduces investigation time and improves workflow efficiency.

## Installation

1. Download the repository
2. Open Chrome Extensions
3. Enable Developer Mode
4. Load unpacked extension
5. Select the folder
